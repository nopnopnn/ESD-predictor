Welcome to ESD-Predictor

ESD-Predictor is a powerful tool designed to predict electrostatic discharge (ESD) parameters, such as peak value and rise rate, based on input current and time data. This tool leverages machine learning models trained on various features extracted from raw electrical data.

The tool allows users to upload CSV files containing time-series data and choose a normalization method (StandardScaler or MinMaxScaler) for feature scaling. After uploading the files, the tool predicts the peak value and rise rate for each file based on the trained model.
